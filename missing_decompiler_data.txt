==============================================================================
SUMMARY OF DECOMPILER GAPS
==============================================================================

1. Data pointers and data table identification: The decompiler doesn't recognize
   the lookup tables at 8422/842B that contain ROM addresses for VRAM data.

2. Palette data: The decompiler marks pal_bg() and pal_spr() as comments,
   suggesting it found palette operations but couldn't identify their purpose.

3. Data fill size/pattern: The vram_fill call shows /* data */ because the
   decompiler can't determine what specific data pattern or size is used.

4. Main loop: The infinite main loop at 860A is not shown because the
   disassembly ends before that point (8000-8D00 range).
